Solución: Números Primos
Explicación Paso a Paso

Para encontrar números primos, iteramos desde el número "desde" hasta "hasta". Para cada número, verificamos si es divisible solo por 1 y por sí mismo. Un número no es primo si encuentra un divisor entre 2 y su raíz cuadrada.

Tips de Optimización

Se puede optimizar el bucle interno para que solo verifique divisores hasta la raíz cuadrada del número. Si no se encuentra un divisor hasta ese punto, no se encontrará después. El uso de `break` detiene la verificación tan pronto como se encuentra un divisor.

Código Funcional
function encontrarPrimos(desde, hasta) {
    let primos = [];
    for (let i = desde; i <= hasta; i++) {
        if (i < 2) continue;
        let esPrimo = true;
        for (let j = 2; j <= Math.sqrt(i); j++) {
            if (i % j === 0) {
                esPrimo = false;
                break;
            }
        }
        if (esPrimo) {
            primos.push(i);
        }
    }
    return primos;
}

Solución: Serie Fibonacci
Explicación Detallada

La serie Fibonacci comienza con 0 y 1. Cada número subsiguiente es la suma de los dos anteriores. Este código utiliza un bucle iterativo que es más eficiente que la recursividad para grandes números.

Código Funcional
function generarFibonacci(limite) {
    if (limite <= 0) return [0];
    let serie = [0, 1];
    while (serie[serie.length - 1] + serie[serie.length - 2] <= limite) {
        let nuevoNumero = serie[serie.length - 1] + serie[serie.length - 2];
        serie.push(nuevoNumero);
    }
    return serie;
}

Solución: Cálculo de IMC
Fórmula Explicada

El IMC se calcula con la fórmula: `peso (kg) / (altura (m) * altura (m))`. Primero, convertimos el peso de libras a kilogramos dividiendo por 2.20462. Luego, aplicamos la fórmula y clasificamos el resultado.

Código Funcional
function calcularIMC(pesoLibras, alturaMetros) {
    const pesoKg = parseFloat(pesoLibras) / 2.20462;
    const altura = parseFloat(alturaMetros);
    const imc = pesoKg / (altura * altura);
    let categoria = '';
    if (imc < 18.5) categoria = 'Bajo peso';
    else if (imc >= 18.5 && imc < 24.9) categoria = 'Normal';
    else if (imc >= 25 && imc < 29.9) categoria = 'Sobrepeso';
    else categoria = 'Obesidad';
    return `Tu IMC es ${imc.toFixed(2)} (${categoria}).`;
}

Solución: Palabra Palíndroma
Explicación Lógica

Una palabra es palíndroma si se lee igual de izquierda a derecha que de derecha a izquierda. Para verificarlo, limpiamos la palabra (a minúsculas y sin espacios), la invertimos y comparamos la original con la invertida.

Código Funcional
function esPalindromo(palabra) {
    const palabraLimpia = palabra.toLowerCase().replace(/\s/g, '');
    const palabraInvertida = palabraLimpia.split('').reverse().join('');
    return palabraLimpia === palabraInvertida;
}
